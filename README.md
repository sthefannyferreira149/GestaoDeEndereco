Este módulo permite que o cliente gerencie seus endereços, possibilitando cadastrar, visualizar, editar e excluir endereços.

O sistema também oferece preenchimento automático de endereço através do CEP, garantindo maior agilidade e precisão durante o cadastro.

Além disso, o administrador possui controle total sobre a base de dados, podendo realizar a manutenção das informações e oferecer suporte direto aos usuários.
